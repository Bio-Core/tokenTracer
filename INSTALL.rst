====================
Installation
===================

1.0 Installation Options
---------------------------

To install the token tracer there are two options:

1. Install from pip
2. Install from git

The first option is preferred for production deployment. The second is preferred for development.

1.1.1 Pip Installation
-----------------------

pip install tokenTracer

Invoke the token tracer with default arguments:

tokenTracer

1.1.2 Git Installation
------------------------

Clone the git repository:

git clone

Run the token tracer found the source with default arguments:

./tokenTracer/tokenTracer/tokenTracer.py
